# Stock Simulation Web (Next.js + Demo Runtime)

태블릿/로컬 환경에서도 바로 확인할 수 있도록, 서버 메모리 기반 데모 런타임을 포함한 모의투자 웹사이트입니다.

## 실행

```bash
npm install
npm run dev
```

## 환경 변수

```bash
YAHOO_FINANCE_TIMEOUT_MS=4000
```

> Supabase 연동으로 확장 가능하지만, 현재 버전은 데모 모드로도 회원가입/로그인/거래/포트폴리오/리더보드가 동작합니다.

## 현재 구현

- 회원가입/로그인 + 세션 쿠키
- 최초 가입시 1,000,000원 지급
- 한국(DB 목록)+미국/API 혼합 검색 (한국 목록은 추후 적재)
- 종목 상세 차트 (1d/5d/1mo, 10초 자동 갱신)
- 매수/매도 거래 규칙
  - 매수: 최소 1,000원, 소수 수량 가능, 수수료 0.1%
  - 매도: 정수 수량만, 수수료 0.1%
- 마이페이지 보유 종목/현금/총자산 (10초 자동 갱신)
- 메인 상단 1위 닉네임 표시

## 주의

데모 런타임은 서버 재시작 시 데이터가 초기화됩니다.
